enrouter is a file based router library for React.

**Please, be aware that the API is not stable yet and can be changed before
reaching v1.0.0.
The author is endeavouring to find the simplest and most elegant API & DX**

## Install

```bash
pnpm add -D enrouter
```

## Requirements

- React
- Vite

  enrouter relies on Vite/Rollup "virtual module" feature for generating routes
  from source modules.
